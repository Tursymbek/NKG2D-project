# Data and code availability

This repository contains custom scripts and derived source-data files for the manuscript **"Toward fratricide-resistant NKG2D receptor-based CAR-T therapy"**.

## What is included

- `scripts/`: custom R scripts recovered from the analysis archive and renamed to match manuscript STAR Methods.
- `analyses/`: generated outputs organized by manuscript STAR Methods section.
- `Supplementary_Materials/`: curated source-data files and selected generated figures organized as Tables S1-S8/Figures S1-S8 where the uploaded archive contained them.
- `Supplementary_Table_Map.csv`: map between manuscript items, scripts, and repository paths.
- `environment/`: session information files recovered from the archive.

## What is not bundled

Large public raw datasets are not included. Download them from UCSC Xena, DepMap/Broad, ENCODE, GENCODE, and GEO using the accessions and identifiers listed in the manuscript KEY RESOURCES TABLE. See `data/external/README.md`.

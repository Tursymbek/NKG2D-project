NKG2D CAR-T supplementary materials

This archive contains the curated source-data tables, rendered supplementary figure files, and figure-reproduction files aligned with the latest manuscript version.

Archive contents:
- Supplementary_Table_Map.csv: file-level index linking each supplementary table or figure item to the corresponding files in this archive.
- Tables S1-S8: non-redundant source-data tables used for the main and supplementary figures and the clinical-trial summary.
- Figures S1-S8: rendered supplementary figure outputs (PNG/PDF where available), R plotting scripts and sessionInfo files where available, or short folder-level README notes when the figure was generated by the main GitHub scripts.

Table folders:
- Table S1: Clinical trial and registry-level source table for NKG2D CAR-T studies in hematologic malignancies and solid tumors, with reference numbers harmonized to the updated bibliography.
- Table S2: TCGA/GTEx tumor-normal tissue statistics and tumor-purity threshold sensitivity.
- Table S3: TCGA/GTEx per-sample composite NKG2DL RNA scores and sample annotations.
- Table S4: DepMap/CCLE NKG2D ligand RNA-expression source data.
- Table S5: DepMap CRISPR/Chronos NKG2D ligand gene-effect source data.
- Table S6: Taiji/STRING/ENCODE promoter-overlap summary source data.
- Table S7: ENCODE TF ChIP-seq accession/provenance tables and acknowledgement text.
- Table S7B: ENCODE ATAC-seq accession/provenance tables and acknowledgement text.
- Table S8: Single-cell CAR-variant, TCAT state, cell-cycle, NKG2DL score, marker-gene, bulk RNA-seq dot-plot and correlation source data.

Supplementary figure folders:
- Figure S1: Sensitivity of tumor-normal composite NKG2DL RNA contrast to TCGA tumor-purity filtering.
- Figure S2: Ligand-level NKG2DL RNA profiles in selected DepMap cancer cell lines.
- Figure S3: DepMap CRISPR dependency summary for NKG2D ligand genes.
- Figure S4: ENCODE promoter-overlap summary for candidate NKG2DL regulators.
- Figure S5: Single-cell CAR-state composition overview from the filtered speedingCAR library.
- Figure S6: Individual NKG2D ligand expression/scores by CAR variant.
- Figure S7: CAR-level summary of functional gene programs and NKG2DL score.
- Figure S8: TCAT program correlations with NKG2D ligand expression.

The archive intentionally excludes raw count matrices, browser screenshots, temporary plotting exports, and hidden/intermediate analysis-history files. Source-data tables are provided in CSV/TSV format; selected ENCODE provenance tables are also provided as XLSX for easier review.
